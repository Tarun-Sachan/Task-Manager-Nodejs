# Task-Manager-Nodejs
  [Live Demo](https://task-manager-nodejs-9xm0.onrender.com/)

 Task Manager MERN app.

 ###Features<br>
 1.User can create and edit tasks.
 2.Data is stored permanently on MongoDB.
 3.API endpoints are managed by using Express.
